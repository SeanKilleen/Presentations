# Cheat Sheet

## The Basics: Help

* `dotnet new /?`
* `dotnet new mvc /?`

## Installing from templates

* `dotnet new [templatename] [options]`
* e.g. `dotnet new sln --name HelloWorld`

## Listing

* `dotnet new --list`

## Installing

### From GitHub

* Clone repo
* `dotnet new -i [path]`
